pygame.joystick

Thin wrapper around the SDL joystick interface. Likely to be changed.
get_count - number of joysticks in system
get_init - query initialization of joystick module
get_name - system name for joystick
init - initialize cdrom module
is_opened - query opened joystick
open - return new joystick object
quit - uninitialize cdrom module
get_count
pygame.joystick.get_count() -> int
    Returns the number of joysticks available on the system. Will be 0 if there are no joysticks.

 
get_init
pygame.joystick.get_init() -> bool
    Returns true when the joystick module is initialized.

 
get_name
pygame.joystick.get_name(id) -> string
    Returns a readable name for the joystick device, given by the system.

 
init
pygame.joystick.init() -> None
    Initialize the joystick module manually

 
is_opened
pygame.joystick.is_opened(id) -> bool
    Returns true if the given joystick id has been previously opened.

 
open
pygame.joystick.open(id) -> Joystick
    Creates a new joystick object for the given joystick id. Once a joystick has been opened, it will start receiving joystick events on the event queue.

 
quit
pygame.joystick.quit() -> None
    Uninitialize the joystick module manually
