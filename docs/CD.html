CD

thin wrapper around the SDL CDROM api, likely to change
eject - ejects cdrom drive
get_cur_frame - query current frame
get_cur_track - query current track
get_empty - query status of drive
get_status - query drive status
pause - pause playing cdrom
play - play music from cdrom
play_tracks - play music from cdrom
resume - resume paused cdrom
stop - stops playing cdrom
eject
CD.eject() -> int
    Ejects the media from the CDROM drive.

 
get_cur_frame
CD.get_cur_frame() -> int
    Get the current frame of a playing CD.

 
get_cur_track
CD.get_cur_track() -> int
    Get the current track of a playing CD.

 
get_empty
CD.get_empty() -> bool
    Returns true when there is no disk in the CD drive.

 
get_status
CD.get_status() -> int
    Get the status of the CDROM drive.

 
pause
CD.pause() -> int
    Pauses the playing CD.

 
play
CD.play(start_frame, nframes) -> int
    Start playing from start_frame, for nframes. If nframes is 0, it will play until the end of the cdrom

 
play_tracks
CD.play_tracks(start_track, start_frame, ntracks, nframes) -> int
    Start playing from start_track, for ntracks and nframes. If ntracks and nframes are 0, it will play until the end of the cdrom

 
resume
CD.resume() -> int
    Resumes playback of a paused CD.

 
stop
CD.stop() -> int
    Stops the playing CD.
