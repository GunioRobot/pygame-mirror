pygame.mask
    pygame module for image masks.
      pygame.mask.pygame.mask.from_surface - Returns the size of the mask.Returns the size of the mask.
      pygame.mask.pygame.Mask - pygame object for representing 2d bitmaskspygame object for representing 2d bitmasks

      pygame.mask.get_at - Returns nonzero if the bit at (x,y) is set.Returns nonzero if the bit at (x,y) is set.
      pygame.mask.set_at - Sets the position in the mask given by x and y.Sets the position in the mask given by x and y.

    Useful for fast pixel perfect collision detection. A Mask uses 1bit per pixel to store which parts collide.

    New in pygame 1.8.

      The overlap tests uses the following offsets (which may be negative):

   +--|B
   |xoffset
